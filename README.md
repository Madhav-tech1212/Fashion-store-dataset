# Retail Fashion Sales Analytics Dashboard

## Business Problem

FashionHub is a multi-brand fashion retailer that sells clothing, footwear, and accessories across multiple seasons.

Although sales remain steady, management has observed several business challenges:

- High product return rates
- Seasonal inventory imbalances
- Heavy markdowns reducing profitability
- Varying customer satisfaction across brands
- Difficulty identifying top-performing products

The executive team requires a Retail Fashion Sales Analytics Dashboard to monitor business performance, understand customer behavior, evaluate pricing strategies, and identify opportunities to improve sales and reduce returns.

## Stakeholders

- CEO
- Sales Manager
- Inventory Manager
- Marketing Team
- Product Manager

## Objectives

- Analyze sales performance
- Identify top-performing brands and categories
- Evaluate markdown effectiveness
- Analyze customer ratings
- Understand return behavior
- Monitor inventory levels
- Generate business recommendations

## Key Performance Indicators (KPIs)

- Total Revenue
- Total Orders
- Average Selling Price
- Average Discount
- Return Rate
- Average Customer Rating
- Inventory Available

## Business Questions

1. Which brands generate the highest revenue?

2. Which categories sell the most?

3. Which season generates the highest sales?

4. Which products receive the highest ratings?

5. Which products have the highest return rate?

6. What are the most common return reasons?

7. Does a higher discount increase sales?

8. Which brands rely heavily on markdowns?

9. Which categories have low inventory?

10. Which products should management promote or discontinue?

## Dataset

This project uses the **Retail Fashion Boutique Data Sales Analytics 2025** dataset from Kaggle.

Source: https://www.kaggle.com/datasets/pratyushpuri/retail-fashion-boutique-data-sales-analytics-2025

The dataset is included in the `data/raw/` folder for convenience.
